# Chat2Poster Documentation

Welcome to the Chat2Poster documentation! Chat2Poster helps you turn AI conversations into beautiful, shareable poster images.

## Quick Links

- [Getting Started](./getting-started) - Install and configure Chat2Poster
- [Themes](./themes) - Customize the look of your posters
- [Export](./export) - Export options and settings

## Features

- **Beautiful Themes** - Dark, Light, and Custom styles
- **Smart Pagination** - Auto-split long conversations
- **One-Click Export** - PNG at 1x/2x/3x DPI
- **Multi-Platform** - ChatGPT, Claude, Gemini support
- **Browser Extension** - Export from any chat page
- **Web App** - Use share links or manual input
