# Chat2Poster 文档

欢迎来到 Chat2Poster 文档！Chat2Poster 帮助你将 AI 对话转换成精美的海报图片。

## 快速链接

- [快速开始](./getting-started) - 安装和配置 Chat2Poster
- [主题](./themes) - 自定义海报外观
- [导出](./export) - 导出选项和设置

## 功能特性

- **精美主题** - 暗色、亮色和自定义样式
- **智能分页** - 自动拆分超长对话
- **一键导出** - 1x/2x/3x 倍率 PNG
- **多平台支持** - ChatGPT、Claude、Gemini
- **浏览器扩展** - 从任意对话页面导出
- **网页版** - 分享链接或手动输入
